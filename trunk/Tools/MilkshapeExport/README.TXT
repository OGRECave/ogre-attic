TO INSTALL MILKSHAPE PLUGIN
---------------------------

After building:

- copy bin\Release\msOgreExporter.dll into the Milkshape folder
- copy Samples\Common\bin\Release\OgreMain.dll into the Milkshape folder

RUNNING THE EXPORTER
--------------------

Select an object in Milkshape and click File|Export|OGRE Mesh / Skeleton...
