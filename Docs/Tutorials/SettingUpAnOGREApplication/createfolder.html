OGRE (Object-Oriented Graphics Rendering Engine)

Creating an application folder

 

This is a pretty trivial step, but worth saying anyway particularly for those who want to follow the OGRE samples closely to make their lives easier.

The Easy Way

The easiest route is to set up your OGRE application in a subdirectory of the 'Samples' directory. Doing this will make getting your compiler / linker settings right later easier because you can just copy them from the other examples.

The Advanced Way

Set up your application folder wherever you like. However, be aware that you will have to be careful when you set up your compiler / linker settings to make sure you reference the OGRE files correctly. Also be aware that your end application will be dependent on OGRE dlls and (possibly) resource files so you will have to copy these manually out of the main OGRE folders in order to ensure your application can reach them all. The detail of this will be covered later on.
